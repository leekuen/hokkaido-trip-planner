① 從 Google 試算表同步

在 Google 試算表編輯行程後，按一下同步即可更新（試算表需設為「知道連結的使用者可檢視」）。

①b 匯入 Excel 行程檔（備用）

也可以直接上傳 .xlsx 檔案，讀取的分頁與同步相同。

② 匯入 Google Maps 已存地點

支援 Google Takeout 匯出的 .json、KML .kml，或含 Title/URL 欄位的 .csv。

不知道怎麼匯出？點我看步驟
  1. 電腦開啟 takeout.google.com（登入你存地點用的 Google 帳號）。
  2. 按「取消全選」，只勾選 Maps（您的地點）。
  3. 建立匯出，等 Google 寄信通知後下載 zip 檔並解壓縮。
  4. 在資料夾裡找 Saved Places.json（或 Takeout/Maps (your places)/ 底下的 json 檔）。
  5. 回到這裡上傳該 json 檔即可。

③ 記帳自動同步到雲端試算表（選填）

設定好之後，新增、編輯、刪除記帳都會同步到 Google 試算表的「記帳」分頁。刪除時是把那一列內容清空（不是整列移除），避免影響其他列的位置。

還沒設定？點我看一次性設定步驟
  1. 打開你的 Google 試算表 → 上方選單「擴充功能」→「Apps Script」。
  2. 把編輯器裡原本的程式碼全部刪掉，貼上下面這段程式碼：
function doPost(e) {
  try {
    var payload = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('記帳');
    var action = payload.action || 'create';
    var d = payload.data || {};

    if (action === 'create') {
      sheet.appendRow([
        '', d.date || '', d.category || '', d.name || '',
        d.ledger || '', d.payer || '', d.method || '',
        d.amountJPY != null ? d.amountJPY : '',
        d.amountTWD != null ? d.amountTWD : '',
        d.note || '', ''
      ]);
      return ContentService.createTextOutput(JSON.stringify({status: 'ok', row: sheet.getLastRow()}))
        .setMimeType(ContentService.MimeType.JSON);
    }

    if (action === 'update') {
      if (!payload.row) throw new Error('缺少 row');
      sheet.getRange(payload.row, 2, 1, 9).setValues([[
        d.date || '', d.category || '', d.name || '', d.ledger || '',
        d.payer || '', d.method || '',
        d.amountJPY != null ? d.amountJPY : '',
        d.amountTWD != null ? d.amountTWD : '',
        d.note || ''
      ]]);
      return ContentService.createTextOutput(JSON.stringify({status: 'ok'}))
        .setMimeType(ContentService.MimeType.JSON);
    }

    if (action === 'delete') {
      if (!payload.row) throw new Error('缺少 row');
      sheet.getRange(payload.row, 1, 1, 11).clearContent();
      return ContentService.createTextOutput(JSON.stringify({status: 'ok'}))
        .setMimeType(ContentService.MimeType.JSON);
    }

    throw new Error('未知的 action: ' + action);
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({status: 'error', message: err.message}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
